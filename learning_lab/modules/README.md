# Module Exercises

Create one folder per learning module. These exercises are intentionally separate
from the production RetailCare architecture.

Naming pattern:

```text
module_01_python_foundations/
module_02_agent_framework/
module_03_tool_calling/
```

Each module folder should contain a short `README.md` explaining:

- what concept is being learned;
- which production files were inspected;
- what scripts or notebooks were created;
- whether anything should be promoted.
